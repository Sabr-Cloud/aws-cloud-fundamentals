# Amazon S3 (Simple Storage Service)
## What S3 is
S3 is an object storage service used to store and retrieve files over the internet.
## Why S3 Is Used
Store Images, Videos and backups
Host static websites
Store logs and data
## key Concepts
Buckets
Objects
Public vs private access
Storage classes
## Real-World Example
A company stores user profile images in S3 instead of on a server.
## What I Learned
S3 is region based 
Buckets must have unique names globally
