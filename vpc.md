# Amazon VPC (Virtual Private Cloud)
## What VPC Is
VPC is a logically isolated network where AWS resources are launched.
## Why VPC Is Used
Control networking
Improve security
Define IP ranges
## Key Concepts
CIDR block
Subnets
Route table
Internet Gateway
## Real-World Example
An EC2 intance in a public subnet is accessible from the internet.
## What I Learned
Subnets belong to only one availability zone
Public subnets need an internet gateway
